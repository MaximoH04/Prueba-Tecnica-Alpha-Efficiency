A high-performance, fully responsive landing page developed for a medical weight loss center. This project prioritizes User Experience (UX) through smooth animations, a mobile-first approach, and specific adaptations for tablet and laptop devices.


## ‚≠ê Key Features


* *100% Responsive & Adaptive Design:*

    * *Mobile (First approach):* Optimized vertical layout with smart content reordering (e.g., placing images between text paragraphs in the "We Can Help" section) for better storytelling.

    * *Tablet (769px - 820px):* Exclusive layout restoration including the "Yellow Background" curve in the hero section, adjusted grid systems (4-column stats), and optimized typography for iPad-like devices.

    * *Desktop:* Spacious layout using CSS Grid and Flexbox for side-by-side content alignment.

* *Advanced UX/UI Enhancements:*

    * *Preloader:* A clean loading screen that ensures all assets are ready before the user interacts, preventing layout shifts (CLS).

    * *Sticky Navbar:* The navigation bar transforms on scroll, acquiring a blurred white background for better visibility.

    * *Scroll Animations:* Elements fade in and slide up gracefully using the Intersection Observer API.

    * *Micro-interactions:* Buttons feature a tactile "press" effect (transform: scale) for   better feedback.

* *Interactive Components:*

    * *Fullscreen Overlay Menu:* A modern, blurred-background mobile menu with a "waterfall" text animation and a hamburger-to-X icon transformation.

    * *Dynamic Countdown Timer:* JavaScript-based timer calculating the remaining time for special offers.

    * *Interactive FAQ:* An accordion-style section with smooth expansion/collapse transitions.

    * *"Back to Top" Button:* A floating button that appears after scrolling, allowing instant navigation to the start.
